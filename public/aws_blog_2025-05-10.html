AWS New Features Update

AWS 新功能更新

Weekly roundup of AWS service enhancements and regional expansions
AWS 服务增强和区域扩展的每周综述

May 10, 2025 · 6 min read
AWS has rolled out an impressive array of service enhancements and regional expansions this week, with significant developments across AI, infrastructure, and database services. In the AI domain, Amazon Q Developer integration with GitHub entered preview, enabling automated feature development, code reviews, and Java modernization within GitHub projects. This integration streamlines the developer experience by allowing teams to assign issues to Amazon Q Developer agents that automatically implement features and run code reviews. Additionally, Bedrock Data Automation now supports audio content extraction, while SageMaker AI enhanced Amazon Q Developer with private code suggestions and workspace context awareness.
AWS 本周推出了一系列令人印象深刻的服务增强和区域扩展，在 AI、基础设施和数据库服务方面都有重大进展。在 AI 领域，Amazon Q Developer 与 GitHub 的集成进入预览版，支持在 GitHub 项目中自动化功能开发、代码审查和 Java 现代化。这种集成通过允许团队将问题分配给 Amazon Q Developer 代理来简化开发人员体验，这些代理会自动实现功能并运行代码审查。此外，Bedrock 数据自动化现在支持音频内容提取，而 SageMaker AI 则增强了 Amazon Q Developer，添加了私有代码建议和工作区上下文感知功能。
AWS continues to innovate across AI, infrastructure, and database services, expanding global reach and enhancing developer productivity.
AWS 继续在 AI、基础设施和数据库服务领域创新，扩大全球覆盖范围并提高开发人员生产力。

GitHub Integration with Amazon Q Developer Preview Available
GitHub 集成 Amazon Q Developer 预览版上线
AWS announces the preview of the Amazon Q Developer integration in GitHub. Developers can use Amazon Q Developer agents for feature development, code review, and Java transformation within GitHub.com and GitHub Enterprise Cloud projects. After installing the Amazon Q Developer application from GitHub, developers can use labels to assign issues to Amazon Q Developer, which will automatically implement new features, generate bug fixes, run code reviews on pull requests, and modernize legacy Java applications.
AWS 宣布 Amazon Q Developer 与 GitHub 集成的预览版。开发人员可以在 GitHub.com 和 GitHub Enterprise Cloud 项目中使用 Amazon Q Developer 代理进行功能开发、代码审查和 Java 转换。安装 GitHub 上的 Amazon Q Developer 应用程序后，开发人员可以使用标签将问题分配给 Amazon Q Developer，它将自动实现新功能、生成错误修复、对拉取请求运行代码审查，并使旧版 Java 应用程序现代化。

Bedrock Data Automation Adds Audio Content Extraction
Bedrock 数据自动化新增音频内容提取功能
Amazon Bedrock Data Automation (BDA) now supports extraction of custom GenAI-powered insights from audio by specifying the desired output configuration through blueprints. Developers can extract custom insights such as summaries, key topics, intents, and sentiment from voice conversations like customer calls, clinical discussions, and meetings. This helps improve employee productivity, reduce compliance costs, and enhance customer experience.
Amazon Bedrock 数据自动化 (BDA) 现在支持通过蓝图指定所需的输出配置，从音频中提取由生成式 AI 驱动的自定义见解。开发人员可以从客户通话、临床讨论和会议等语音对话中提取摘要、关键主题、意图和情感等自定义见解。这有助于提高员工生产力，降低合规成本，并增强客户体验。

SageMaker AI Enhances Amazon Q Developer with Custom Code Suggestions
SageMaker AI 增强 Amazon Q Developer 添加私有代码建议功能
Amazon SageMaker AI introduces significant enhancements to Amazon Q Developer in Jupyter Lab, adding customization of code suggestions based on private code repositories and the ability to include entire workspace context for improved code assistance. The new customization feature enables Amazon Q Developer to assist with software development that conforms to your team's internal libraries, proprietary algorithmic techniques, and enterprise code style.
Amazon SageMaker AI 为 Jupyter Lab 中的 Amazon Q Developer 引入了重要增强功能，增加了基于私有代码库的代码建议定制化，以及包含整个工作区上下文以改进代码辅助的能力。新的定制功能使 Amazon Q Developer 能够协助符合团队内部库、专有算法技术和企业代码风格的软件开发。

SageMaker HyperPod Integrates with EventBridge for Status Change Events
SageMaker HyperPod 集成 EventBridge 实现状态变更通知
Amazon SageMaker HyperPod now integrates with Amazon EventBridge, enabling near-real time notifications about changes in cluster status. SageMaker HyperPod delivers two types of notifications: Cluster Status Change Events that notify you when your HyperPod cluster transitions between states, and Node Health Events that notify you when nodes change health status or are automatically replaced during recovery from failures.
Amazon SageMaker HyperPod 现已与 Amazon EventBridge 集成，实现集群状态变更的近实时通知。SageMaker HyperPod 提供两种类型的通知：集群状态变更事件（在 HyperPod 集群状态转换时通知您）和节点健康事件（在节点健康状态变更或在故障恢复期间自动替换时通知您）。

Amazon ECS Adds One-Click Service Deployment Rollbacks
Amazon ECS 新增一键式服务部署回滚功能
Amazon Elastic Container Service (Amazon ECS) announced a new feature that allows you to easily rollback your Amazon ECS service to a previous safe state if a deployment fails. With the new stopDeployment API action, ECS automatically rolls back the service to the last service revision that reached steady state.
Amazon Elastic Container Service (Amazon ECS) 宣布推出一项新功能，允许您在部署失败时轻松将 Amazon ECS 服务回滚到先前的安全状态。通过新的 stopDeployment API 操作，ECS 会自动将服务回滚到上一个达到稳定状态的服务修订版本。
